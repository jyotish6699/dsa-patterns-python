🎯 Problem: Longest Consecutive Sequence

-You are given an unsorted array of integers.

👉 Your task is to find the length of the longest sequence of consecutive numbers (numbers that follow each other with +1 difference).

📌 Important Rules:
1. The numbers do not need to be next to each other in the array
2. Order in array does NOT matter
3. You only care about values forming a continuous chain


🧠 What You Will Learn From THIS ONE Question
1. Pattern: sequence detection using set
2. Skill: avoid redundant computation
3. Thinking: when NOT to run loop
4. Concept: amortized O(n) thinking
5. Insight: local expansion strategy
